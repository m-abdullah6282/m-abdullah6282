## 🚀 Featured Projects

### 🤖 [Competitor Intel Agent](https://github.com/m-abdullah6282/Competitor-Intel-Agent)
Autonomous, multi-agent pricing-intelligence engine that scouts and analyzes competitor pricing on a schedule. A **Scout** agent scrapes and compresses raw HTML with BeautifulSoup before inference to cut LLM token costs, an **Analyst** agent interprets the changes, and the whole flow is orchestrated with **LangGraph** behind a **FastAPI** service. Uses an asynchronous polling pattern to sidestep API timeouts and persists historical price changes to **PostgreSQL** via SQLAlchemy.

`Python` · `FastAPI` · `LangGraph` · `SQLAlchemy` · `PostgreSQL` · `BeautifulSoup`

### 📊 [A/B Testing: CTR Analysis](https://github.com/m-abdullah6282/ab-testing-ctr-analysis)
Statistical hypothesis-testing framework that evaluates whether a UI change meaningfully improves Click-Through Rate. Runs a two-sample Z-test for proportions at 95% confidence, reports p-values and confidence intervals, and checks results against a practical-significance threshold — not just statistical significance.

`Python` · `Pandas` · `NumPy` · `SciPy` · `Matplotlib`
